About Me

I'm a third‑year Computer Engineering student at UofT with a strong interest in hardware‑software integration and user‑focused systems. Recently, I led development on CHUMMY, an interactive music‑therapy system for dementia care: Raspberry Pi Zero + Python audio stack on device, a Node/Express + WebSockets backend, and a React caregiver dashboard with live status and BPSD logging.

Beyond CHUMMY, I’ve built a two‑octave Verilog synthesizer (PS/2 keyboard, Quartus/ModelSim), a VGA “Whack‑a‑Mole” game on DE1‑SoC, a GIS map app (C++ pathfinding/A* & multi‑threaded courier routing), and contributed to EKF unit tests in C++ for a driverless racing team.

I love learning by building, shipping, and iterating — and I’m actively looking for co‑op opportunities where I can contribute to real products while leveling up in embedded, systems, and frontend engineering.
